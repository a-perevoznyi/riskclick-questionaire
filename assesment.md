# Risklick Technical Assessment Package

This package contains a ready-to-upload version of the technical assessment in Markdown format.

### Files

* `assesment.md` — final answer text
* `code-snippets/pricing-config.ts` — pricing configuration loading and validation example
* `code-snippets/resolve-subscription-plan.ts` — subscription plan resolution example

### Notes

* The project example is based on a real Shopify SaaS case using Remix, React, Node.js, AWS CloudFront, and CI/CD.
* The code examples are intentionally simplified, but written in a cleaner and more enterprise-style way.
* The code is meant to explain design thinking and implementation approach, not to represent the full production codebase.
